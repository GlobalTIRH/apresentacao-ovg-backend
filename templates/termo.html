TERMO DE CONSENTIMENTO PARA TRATAMENTO DE DADOS PESSOAIS

Este termo de consentimento é fornecido pela Global TI+RH Solutions, com sede em Goiânia, Goiás, Brasil, e tem como objetivo informar e obter o consentimento do titular dos dados pessoais para o tratamento de suas informações conforme descrito abaixo.

  1. Finalidade da Coleta: O presente termo tem como objetivo registrar a manifestação livre, informada e inequívoca do(a) titular dos dados pessoais, para que seus dados sejam coletados, tratados e armazenados pela Global TI+RH Solutions, conforme as finalidades abaixo:
    • Cadastro e identificação do usuário.
    • Comunicação com o titular por meio de envio de email com os dados das vagas, conforme interesse manifestado pelo usuário.
  2. Dados Coletados: Os dados pessoais que poderão ser coletados incluem, mas não se limitam a:
    • Nome completo
    • E-mail
    • Telefone
  3. Compartilhamento de Dados: Seus dados poderão ser compartilhados com terceiros apenas quando necessário para a execução dos serviços contratados, cumprimento de obrigações legais ou com empresas parceiras, sempre respeitando os princípios da LGPD.
  4. Armazenamento e Segurança: Os dados serão armazenados de forma segura, com medidas técnicas e administrativas para proteger contra acessos não autorizados, vazamentos e incidentes de segurança.
  5. Direitos do Titular: O(a) titular dos dados poderá, a qualquer momento, solicitar:
    • Acesso aos dados.
    • Correção de dados incompletos, inexatos ou desatualizados.
    • Anonymização, bloqueio ou eliminação de dados desnecessários.
    • Revogação deste consentimento.
  6. Consentimento: Declaro que li e compreendi os termos acima e autorizo, de forma livre e consciente, o tratamento dos meus dados pessoais conforme descrito neste documento.
